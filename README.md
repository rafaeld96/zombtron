# Zombtron
A game where the player is faced against onslaughts of zombies. Inspiration is taken from the legacy game "Robotron".

# Instructions

Go to the root folder. Run the command

```
javac zombtron/*.java
java zombtron.Zombtron
```

You may now play the game.
